# מונדיאלה נבדלה — מהדורת נוקאאוט (Knockout Edition)

**תאריך:** 2026-06-28
**סטטוס:** ספק מאושר עקרונית, לקראת תוכנית ביצוע
**קובץ יעד:** `index.html` (קובץ יחיד), `manifest.json` לפי הצורך

## הקשר ובעיה

האפליקציה (`mundial-companion`, LIVE ב-mundial.karin-arad.com) בנויה כולה סביב **שלב הבתים**:
- `FIXTURES` = 72 משחקי הבתים, נגמרים ב-28.6.2026.
- `GROUPS` = 12 בתים, 48 נבחרות, גריד צבעוני.
- הערת "משחקים במקביל" (שערוריית חיחון 1982) — רלוונטית רק לסיבוב האחרון בבתים.
- הפרומפטים והמילון מנוסחים סביב "איזה משחק בית כדאי לראות".

**באג קריטי:** כש-`todayMatches()` מחזיר ריק (מ-29.6 ואילך), `renderToday()` נופל ל-`FIXTURES.slice(0,2)` ומציג את **משחק הפתיחה מ-11.6**. כלומר מ-29.6 האפליקציה מציגה תוכן שגוי.

שלב הבתים נגמר, התחיל הנוקאאוט. צריך לרענן תוכן, מבנה תצוגה, ונראות — **בלי לשנות טכנולוגיה**.

## עקרון על

נשמרת כל הסטאק: קובץ HTML יחיד, vanilla JS, תלת-לשוני (he/en/de) עם `T[LANG]`/`data-i18n`, דיפלוי בוורסל, PWA (`sw.js`/`manifest.json`), וכל כפתור פותח את ChatGPT דרך `ask()` עם פרומפט מודע-תאריך. משנים רק: דאטה, מבני תצוגה, נראות, ניסוחים.

## החלטות שהתקבלו (brainstorm)

1. **נתוני ברקט:** מבנה קבוע + הצ'אט ממלא. נקודד לו"ז ושלבים שידועים מראש (פיפ"א); שמות הנבחרות מתמלאים ככל שנסגרים, והשאר מתווית-סלוט + השלמה חיה מהצ'אט.
2. **קטע מרכזי:** שומרים את גריד הבתים כרפרנס + מוסיפים מקטע ברקט מעליו.
3. **נראות:** אבולוציה — שומרים זהות, מוסיפים שכבת "דרמת נוקאאוט".
4. **מודחים:** מסומנים בגריד הבתים (עדכון ידני קליל).
5. **ברקט רספונסיבי:** אותה דאטה, שתי צורות — עץ אופקי בדסקטופ, אקורדיון אנכי במובייל.

## ארכיטקטורת תוכן

### א. דאטה — מערך `KNOCKOUT` חדש

לצד `FIXTURES` (שנשאר לרפרנס היסטורי). כל משחק:

```js
{
  round:'R32',                  // R32 | R16 | QF | SF | THIRD | FINAL
  matchNo:73,                   // מספר משחק רץ (פיפ"א)
  date:'2026-06-29', time:'22:00', venue:'...',   // שעון ישראל, מומר לשעון מקומי כמו היום
  home:{team:'מקסיקו', slot:'1A'},    // team=null ⇒ מציגים את ה-slot
  away:{team:null,      slot:'3B/E/F/I'}
}
```

- `slot` = תווית מתורגמת ("מנצחת בית A", "אחת השלישיות הטובות"); כש-`team` ידוע — מציגים שם + דגל.
- **המבנה (תאריכים/אצטדיונים/שלבים/סלוטים) קבוע ומאומת מול פיפ"א** לפני קידוד — לא מהזיכרון (ע"פ העדפת אימות-במקור).
- שלבים: סבב ה-32 → שמינית גמר → רבע גמר → חצי גמר → הקרב על המקום ה-3 → גמר.

### ב. "מה רואים עכשיו"

- מקור הנתונים עובר מ-`FIXTURES` ל-`KNOCKOUT` (לוגיקת `todayMatches`/`tomorrowMatches`/`fxTime`/timezone — ללא שינוי, רק המקור).
- **תיקון fallback:** במקום `FIXTURES.slice(0,2)` → "בין שלבים — המשחק הבא:" + המשחק הקרוב ב-`KNOCKOUT`, או הודעה אם אין.
- שורת "המשחקים של מחר" נשארת.

### ג. מקטע ברקט חדש (מעל הבתים)

עץ הדרך לגמר. כל קלף משחק: דגלים+שמות או תווית-סלוט, שעה מקומית, אצטדיון, ושני כפתורים (`כל המידע` / `וואסח`) — אותו `matchAsk`/`matchWasach` הקיים.

**רספונסיביות (אותה דאטה, שתי צורות):**
- **דסקטופ:** עץ-ברקט אופקי עם קווי מחבר בין השלבים, מותאם RTL.
- **מובייל:** אקורדיון אנכי לפי שלב — הסבב הקרוב פתוח, השאר מתקפלים.
- מומש ב-CSS responsive (media query) על אותו DOM, ללא כפילות דאטה.

### ד. הבתים — רפרנס היסטורי

- גריד `GROUPS` נשאר, ממותג מחדש ("מאיפה כולם הגיעו"), **מקופל כברירת מחדל** מתחת לברקט.
- כפתורי הנבחרות נשארים.
- **סימון מודחים:** שדה `out:true` לכל נבחרת מודחת ב-`GROUPS` → סטייל מעומעם (`opacity`, אולי קו חוצה) + תג "הודחה". עדכון ידני קל ככל שמתקדם הנוקאאוט.

### ה. פרומפטים ומילון

- **פרומפטים חדשים** (מחליפים את ה"איזה משחק בית"): "מי הפייבוריט בדרך לגמר", "הדרמה של הערב — מה על הכף", "תרחישי הברקט", "מי כבר הודח ומי שרד".
- **מילון** מתווסף: נוקאאוט, שמינית/רבע/חצי גמר, הארכה, פנדלים (דו-קרב 11 מ'), ברקט, "אין מחר".
- **כל מחרוזת חדשה בשלוש השפות** (he/en/de) — חובה, אחרת תפר את מתג השפה.

### ו. הסרות

- `simulNote()` + מחרוזות `simulTitle`/`simulBody` (he/en/de) + הקריאה ב-`rowsWithNote` — נמחק.
- ניסוחי "משחק בית" בפרומפטים/כותרות — מוחלפים.

## נראות (אבולוציה)

נשמרים: פלטה ורודה-סגולה, בועות מונפשות, Rubik/Baloo 2, פינות מעוגלות.
מתווסף — שכבת "דרמת נוקאאוט":
- אקסנט אדום-הדחה (משתנה CSS חדש) לרגעי הכרעה ולתגי "הודחה".
- קווי מחבר של ברקט.
- סרט קטן בהירו: "⚔️ נוקאאוט · אין מחר".
- ה-CTA בהירו מתעדכן לתקציר שלב נוקאאוט.
- וריאנט כפתור "הדחה" אדום ל-`chatpill` ברגעי הכרעה.
